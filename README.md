## 注意

- iOS 6時代に**Objective-C**で書いたサンプルなため、より良い解決方法があると思います。

## 環境

- Xcode 13.4.1
    - 一部UIの崩れはありますが、そのままでビルド可能になっています。

## 内容

- 最小限のコードでの描画
- 文字属性の設定
- 日本語行間問題の解決
- タップ情報の取得
- タップ範囲をViewで囲う
- 1行ずつ描画
- 1文字ずつアニメーションで描画
- 縦書
- sizeToFit
